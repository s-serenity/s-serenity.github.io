第二十一篇 java(2)- 集合框架

集合框架

java集合框架包括 Collection 和 Map 两种，Collection 存储着对象的集合，而 Map 存储着键值对(两个对象)的映射表。

Collection

Collection下包括Set、List、Queue，各自又包含了使用不同方式的实现。

Set

Set下包括TreeSet，HashSet，LinkedHashSet。其中TreeSet基于红黑树实现。
